# battle-game
A simple RPG battle game similar to a Pokemon Go Gym battle.

Both fighters start out with 100 life points. Every time you battle, your strength is based off a random number. If your strength is greater then your opponents, then he loses life points. If his strength is greater then yours then you lose life points.

Includes a Player class (or Pokemon) with two objects of that class: once for the user, once for the computer.  

Some attributes about the class include:

life points (hp)
strength
generator for a random number (make this static)

Also includes a default constructor (no parameters) to set the life points(hp) and strength
Behaviors include:

battle
receive damage

Plus a few accessor methods for the strength and life points.
